# AOL HCI - Fashion Brand Homepage

**Academic Online Learning (AOL) — Human-Computer Interaction**
*HTML | CSS | Web Design*

## Overview

This project is a **fashion brand homepage** designed as part of the AOL Human-Computer Interaction course. The website showcases a luxury fashion brand called **Christian Wijaya** with a focus on elegant design, user experience, and responsive layout.

## Features

| Feature | Description |
|---------|-------------|
| **Hero Section** | Full-screen video background with overlay text |
| **Navigation** | Clean top navigation bar with category links |
| **Product Categories** | Men's Wear, Women's Wear, Jewelry, Purse sections |
| **Limited Edition** | Highlighted featured collection |
| **Footer** | Multi-column footer with customer service links |

## Design Elements

- **Typography**: Montserrat font family
- **Color Palette**: Warm neutral tones (cream, beige, black)
- **Layout**: CSS Flexbox-based responsive design
- **Media**: Video backgrounds and product imagery

## What I Learned

- **HTML5** semantic structure and multimedia integration
- **CSS3** Flexbox layout and responsive design
- **UI/UX Principles** for luxury fashion branding
- **Figma** prototyping and design system
- **Visual hierarchy** and typography best practices

## Note

This project was submitted as part of the AOL (Academic Online Learning) assessment for the Human-Computer Interaction course.
